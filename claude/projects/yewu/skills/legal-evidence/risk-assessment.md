# 五维风险评估模型

## 评估框架

从5个维度对案件进行量化风险评估，加权得出总分。

| 维度 | 权重 | 评估内容 |
|------|------|---------|
| **证据风险** | 30% | 证据是否充分、是否有原件、是否能形成完整链条 |
| **法律风险** | 25% | 法律依据是否明确、是否有争议、是否有不利判例 |
| **对方风险** | 20% | 对方可能的抗辩、反诉、调解策略 |
| **执行风险** | 15% | 对方是否有履行能力、是否有可执行财产 |
| **时间风险** | 10% | 诉讼周期、时效、审限 |

## 评分标准

### 证据风险（30%）

| 分数 | 标准 |
|------|------|
| 90-100 | 证据充分，原件齐全，链条完整 |
| 70-89 | 证据基本充分，部分瑕疵可补强 |
| 50-69 | 证据有缺口，需要补强 |
| 30-49 | 证据严重不足，关键要件缺证据 |
| 0-29 | 几乎无证据支撑 |

### 法律风险（25%）

| 分数 | 标准 |
|------|------|
| 90-100 | 法律依据明确，无争议 |
| 70-89 | 法律依据基本明确，有少量争议 |
| 50-69 | 法律适用有争议，需检索判例 |
| 30-49 | 法律适用重大争议，不利判例较多 |
| 0-29 | 法律依据不明确或对我方不利 |

### 对方风险（20%）

| 分数 | 标准 |
|------|------|
| 90-100 | 对方抗辩理由弱，无反诉可能 |
| 70-89 | 对方有一定抗辩理由，但可应对 |
| 50-69 | 对方抗辩理由较强，需要充分准备 |
| 30-49 | 对方可能反诉，我方需同时应对 |
| 0-29 | 对方抗辩/反诉理由很强，我方被动 |

### 执行风险（15%）

| 分数 | 标准 |
|------|------|
| 90-100 | 对方有充足财产，执行无障碍 |
| 70-89 | 对方有一定财产，执行基本可行 |
| 50-69 | 对方财产状况不明，需调查 |
| 30-49 | 对方可能无财产，执行困难 |
| 0-29 | 对方已有多起被执行案件，执行极困难 |

### 时间风险（10%）

| 分数 | 标准 |
|------|------|
| 90-100 | 事实清楚，可适用简易程序 |
| 70-89 | 普通程序，但周期可控 |
| 50-69 | 可能涉及鉴定/审计，周期较长 |
| 30-49 | 案件复杂，周期很长 |
| 0-29 | 可能涉及二审/再审，周期极长 |

## 输出示例

```
## 案件风险评估报告

### 总分：68/100（中等风险）

### 各维度得分

| 维度 | 得分 | 说明 |
|------|------|------|
| 证据风险 | 75/100 ⚠️ | 送货单有签收但无原件 |
| 法律风险 | 85/100 ✅ | 法律依据明确，无争议 |
| 对方风险 | 60/100 ⚠️ | 对方可能主张质量问题抗辩 |
| 执行风险 | 50/100 🔴 | 对方公司已有多起被执行案件 |
| 时间风险 | 70/100 ⚠️ | 普通程序审限6个月 |

### 重点风险提示

1. **执行风险较高**：建议诉前保全，冻结对方账户
2. **证据需补强**：送货单原件丢失，需收集其他佐证

### 建议

- 诉前申请财产保全
- 补强送货证据（对账单、付款记录）
- 准备应对质量问题抗辩
```

## 使用场景

- **接案决策**：是否接受委托、风险代理是否可行
- **预期管理**：给当事人讲清楚风险点
- **策略调整**：哪个维度风险高，重点补强哪个
- **收费参考**：风险越高，律师费相应调整
