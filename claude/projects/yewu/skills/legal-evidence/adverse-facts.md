# 不利事实分析

## 什么是不利事实

案件中对我方不利的事实，可能被对方利用来攻击我方主张。

## 8类不利事实

### 1. 自认事实

**定义**：我方当事人在书面或口头陈述中承认了对己方不利的事实

**示例**：
- 微信中说"货确实有点瑕疵"
- 邮件中说"我们确实延期了"
- 电话录音中承认部分欠款

**严重程度**：🔴 高

**应对策略**：
1. 解释为"商业客套"而非法律承认
2. 强调不影响合同目的实现
3. 补充证据反驳

### 2. 矛盾证据

**定义**：我方两份或多份文件的说法不一致

**示例**：
- 合同约定30天交货，但送货单显示45天才交
- 发票金额与合同金额不一致

**严重程度**：🔴 高

**应对策略**：
1. 解释矛盾原因（笔误、计算方式不同等）
2. 提供其他证据佐证正确的说法
3. 主张以哪份为准

### 3. 缺失证据

**定义**：关键事实没有书面证据

**示例**：
- 口头约定没有书面记录
- 交付货物没有签收单

**严重程度**：🟠 中

**应对策略**：
1. 收集间接证据（聊天记录、付款记录）
2. 申请法院调查取证
3. 利用对方自认

### 4. 不利书证

**定义**：合同或其他书面文件中有对我方不利的条款

**示例**：
- 合同约定"逾期交货视为根本违约"
- 合同约定"买方有权拒收任何瑕疵货物"

**严重程度**：🟠 中

**应对策略**：
1. 主张格式条款无效（民法典第497条）
2. 主张该条款不适用于本案情况
3. 强调对方未实际行使该权利

### 5. 时效风险

**定义**：催款间隔过长，可能被抗辩时效

**示例**：
- 最后一次催款是3年前
- 没有书面催款记录

**严重程度**：🟠 中

**应对策略**：
1. 收集中断时效的证据（催款函、对方承诺还款）
2. 主张时效起算点晚于对方主张
3. 适用特殊时效规定

### 6. 程序瑕疵

**定义**：我方在程序上有过错

**示例**：
- 未在举证期限内提交证据
- 未按约定发送解除通知

**严重程度**：🔴 高

**应对策略**：
1. 申请法院采纳逾期证据
2. 主张程序瑕疵不影响实体权利
3. 寻找补救措施

### 7. 对方有利证据

**定义**：对方持有对我方不利的证据

**示例**：
- 对方有我方签字的验收单
- 对方有我方确认的对账单

**严重程度**：🟠 中

**应对策略**：
1. 质疑证据真实性
2. 解释签字背景（被迫、误解等）
3. 提供反证

### 8. 证人风险

**定义**：关键证人可能不出庭或改口

**示例**：
- 证人是对方员工
- 证人与对方有利害关系

**严重程度**：🟡 低

**应对策略**：
1. 申请法院通知证人出庭
2. 准备书面证言
3. 收集其他证据佐证

## 输出格式

```
## 不利事实清单

### 🔴 高风险

**不利事实1：[描述]**
- 严重程度：🔴 高
- 影响：[对方可能如何利用]
- 应对策略：
  1. [策略1]
  2. [策略2]
  3. [策略3]

### 🟠 中风险

**不利事实2：[描述]**
...

### 🟡 低风险

**不利事实3：[描述]**
...
```

## 使用场景

- **庭前准备**：提前知道对方会攻击什么，准备好应对
- **预期管理**：告诉当事人"这个问题存在，但有应对办法"
- **代理词预埋**：在代理词中主动解释不利事实，不让对方先发制人
- **调解策略**：知道弱点在哪里，调解时心里有数
