# 代理词模板

## 文书结构

```
代 理 词

审判长、审判员：

[律师事务所名称]依法接受[原告/被告名称]的委托，指派[律师姓名]律师担任[原告/被告名称]与[对方名称][案由]纠纷一案的诉讼代理人。本代理人根据本案事实和法律规定，发表如下代理意见，请合议庭予以采纳。

一、[核心观点1——事实认定]

[论述]

二、[核心观点2——法律适用]

[论述]

三、[核心观点3——责任承担]

[论述]

四、[回应对方观点（如有）]

[论述]

综上所述，代理人认为[总结]，请求贵院依法支持[原告/被告]的诉讼请求/驳回[原告]的诉讼请求。

以上代理意见，请合议庭予以采纳。

[律师事务所名称]
代理人：[律师姓名]
[日期：年 月 日]
```

## 关键规范

### 格式要求

- 开头："审判长、审判员："
- 结尾：落款律所名称 + 代理人姓名 + 日期
- 不需要"此致"和法院名称

### 内容策略

- **观点明确**：每个大点用一句话概括核心观点
- **论据充分**：事实 + 法条 + 逻辑推理
- **回应对方**：针对对方的答辩/质证意见逐一回应
- **总结有力**：最后一段简明扼要总结

### 与起诉状/答辩状的区别

| | 起诉状/答辩状 | 代理词 |
|--|--------------|--------|
| 格式 | 公文格式 | "审判长、审判员" |
| 法条 | 宜少不宜多 | 可以详细引用（精准到款、项） |
| 篇幅 | 简洁 | 可以展开论述 |
| 时机 | 立案时/答辩期提交 | 庭前或庭后提交 |
