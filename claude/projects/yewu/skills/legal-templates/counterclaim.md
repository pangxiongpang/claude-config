# 民事反诉状模板

## 文书结构

```
民事反诉状

反诉原告（本诉被告）：[名称/姓名]
住所地：[地址]
法定代表人/负责人：[姓名]，[职务]
委托诉讼代理人：[姓名]，[律所名称]律师

反诉被告（本诉原告）：[名称/姓名]
住所地：[地址]

反诉请求：

一、判令反诉被告[具体请求]；
二、判令反诉被告承担本案反诉部分全部诉讼费用。

（反诉标的额：人民币[金额]元）

事实与理由：

[当事人关系/合同签订情况]

[合同核心条款]

[合同履行情况]

[违约事实/争议事实]

[损失情况]

综上所述，反诉原告依据《中华人民共和国民法典》第XXX条等相关规定，提起反诉，望判如所请！

此致
[法院名称]

反诉人：[反诉人名称]
[日期：年 月 日]
```

## 关键规范

### 当事人标注

- 必须标注"反诉原告（本诉被告）"和"反诉被告（本诉原告）"
- 明确反诉与本诉的当事人对应关系

### 标的要求

- 反诉标的额单独标注
- 如本诉与反诉标的可合并，注明合计金额

### 反诉条件

- 反诉与本诉有牵连关系
- 反诉不属于其他法院专属管辖
- 反诉在本诉受理后、法庭辩论终结前提出

### 注意事项

- 反诉状单独提交，不附在答辩状中
- 反诉费单独计算和预交
- 反诉请求与本诉请求应明确区分
