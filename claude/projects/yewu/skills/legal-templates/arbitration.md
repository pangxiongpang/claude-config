# 商事仲裁申请书模板

## 文书结构

```
仲裁申请书

申请人：[名称/姓名]
住所地：[地址]
法定代表人：[姓名]，[职务]
委托代理人：[姓名]，[律所名称]律师

被申请人：[名称/姓名]
住所地：[地址]

仲裁申请事项：

一、裁令被申请人[具体请求]；
二、裁令被申请人承担本案全部仲裁费用。

（申请标的额：人民币[金额]元）

事实与理由：

[当事人关系/合同签订情况]

[合同核心条款]

[合同履行情况]

[违约事实/争议事实]

[催告/协商情况]

[如有仲裁条款：申请人与被申请人签订的《XX合同》第X条约定，因本合同引起的争议提交XX仲裁委员会仲裁。]

综上所述，申请人依据《中华人民共和国民法典》第XXX条等相关规定，申请仲裁，望裁如所请！

此致
[仲裁委员会名称]

申请人：[申请人名称]
[日期：年 月 日]
```

## 术语转换规则

从起诉状转为仲裁申请书时，需做以下替换：

| 起诉状用语 | 仲裁申请书用语 |
|-----------|---------------|
| 起诉状 | 仲裁申请书 |
| 原告 | 申请人 |
| 被告 | 被申请人 |
| 诉讼请求 | 仲裁申请事项 |
| 诉讼费 | 仲裁费 |
| 此致XX人民法院 | 此致XX仲裁委员会 |
| 具状人 | 申请人 |
| "诉至贵院" | "申请仲裁" |
| "望判如所请" | "望裁如所请" |

## 注意事项

- **仲裁条款**：如有仲裁条款，在事实段中注明
- **仲裁机构**：确认仲裁委员会全称
- **一裁终局**：提醒当事人仲裁一裁终局，不可上诉
- **仲裁费**：仲裁费标准与诉讼费不同，需另行计算
