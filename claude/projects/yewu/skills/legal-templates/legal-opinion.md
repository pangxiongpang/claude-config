# 法律意见书模板

## 文书结构

```
法律意见书

编号：[编号]

致：[委托人名称]

[律师事务所名称]（以下简称"本所"）依法接受[委托人名称]（以下简称"委托人"）的委托，就[事项名称]出具本法律意见书。

一、出具依据

本所依据以下文件材料和法律规范出具本法律意见书：

（一）文件材料
1. [文件名称1]
2. [文件名称2]
...

（二）法律规范
1. 《中华人民共和国民法典》
2. [其他相关法律法规]
...

二、基本事实

[按时间顺序陈述已确认的基本事实]

三、法律分析

（一）关于[第1个法律问题]

依据《XXX》第X条规定，[法条内容]。

本所认为，[法律分析]。

（二）关于[第2个法律问题]

[同上格式]

...

四、结论与风险提示

综上所述，本所认为：

1. [结论1]
2. [结论2]
...

风险提示：
1. [风险1]
2. [风险2]
...

五、声明

本法律意见书仅基于委托人提供的文件材料出具，如委托人提供的材料不完整或与事实不符，可能影响本意见书的结论。本意见书仅供委托人参考，不作为任何第三方的决策依据。

[律师事务所名称]
承办律师：[律师姓名]
[日期：年 月 日]

[律所盖章]
```

## 关键规范

### 适用场景

- IPO/债券发行
- 并购重组
- 私募备案
- 股权激励
- 重大交易的法律风险评估

### 出具依据

- 列明所有依据的文件材料
- 列明所有依据的法律规范
- 确保文件材料完整

### 法律分析

- **先法条再分析**：先引用法条，再进行分析
- **逐项分析**：每个法律问题单独分析
- **逻辑清晰**：事实 → 法条 → 分析 → 结论

### 风险提示

- 明确指出可能存在的风险
- 风险等级可分级（高/中/低）
- 建议应对措施

### 声明

- 说明意见书的局限性
- 明确责任边界
- 防止被不当使用
