# 民事上诉状模板

## 文书结构

```
民事上诉状

上诉人（原审×告）：[名称/姓名]
住所地：[地址]
法定代表人/负责人：[姓名]，[职务]
委托诉讼代理人：[姓名]，[律所名称]律师

被上诉人（原审×告）：[名称/姓名]
住所地：[地址]

上诉人因与被上诉人[案由]纠纷一案，不服[法院名称]（XXXX）XX民初XX号民事判决/裁定，现提起上诉。

上诉请求：

一、撤销[法院名称]（XXXX）XX民初XX号民事判决第X项；
二、依法改判[具体请求]；
三、本案一、二审诉讼费用由被上诉人承担。

上诉理由：

一、一审判决认定事实错误：[具体指出]

二、一审判决适用法律错误：[具体指出]

三、[其他上诉理由]

综上所述，[法院名称]（XXXX）XX民初XX号民事判决认定事实不清/适用法律错误，请求贵院依法改判。

此致
[二审法院名称]

上诉人：[上诉人名称]
[日期：年 月 日]
```

## 关键规范

### 格式要求

- **公文格式**，不适用三段式
- 上诉请求明确：撤销/改判范围精确到项
- 上诉理由分层：事实错误 → 法律错误 → 程序错误

### 上诉请求

- **撤销范围**：明确撤销判决书的哪一项
- **改判内容**：具体写明希望如何改判
- **诉讼费**：一并请求

### 上诉理由

- **事实错误**：指出一审认定事实的错误之处
- **法律错误**：指出一审适用法律的错误之处
- **程序错误**：指出一审程序违法之处（如有）

### 时效

- **判决上诉**：收到判决书之日起15日内
- **裁定上诉**：收到裁定书之日起10日内
- **逾期后果**：丧失上诉权

### 注意事项

- 附一审判决书/裁定书复印件
- 上诉状提交份数：按对方当事人人数+1份
- 上诉费在提交上诉状时预交
