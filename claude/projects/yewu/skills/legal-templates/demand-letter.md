# 律师函模板

## 文书结构

```
律师函

[编号]：[律所简称]函字（XXXX）第XX号

致：[收件人全称]

[律师事务所名称]（以下简称"本所"）依法接受[委托人全称]（以下简称"委托人"）的委托，指派[律师姓名]律师（执业证号：[号码]）就[事由]相关事宜，特致函如下：

一、委托声明

本所受委托人委托，就[简述委托事项]之相关法律问题，郑重致函贵方。

二、基本事实

[按时间顺序客观陈述事实经过]

三、法律依据

依据《中华人民共和国民法典》第XXX条[及其他相关法律条文]的规定，[法律分析]

四、要求

基于上述事实和法律，委托人要求贵方：

1. [具体第1项要求，包括金额、期限]
2. [具体第2项要求]
...

五、后果告知

如贵方在上述期限内未能满足委托人的合理要求，委托人将依法采取[诉讼/仲裁/其他法律手段]等措施追究贵方的法律责任，届时贵方将可能承担额外的诉讼费、律师费及违约损失等。

专此函告，望贵方慎重考虑！

[律师事务所名称]
承办律师：[律师姓名]
[日期：年 月 日]

[律所盖章]
```

## 关键规范

### 编号规则

- 统一格式：律所简称 + 年度 + 序号
- 例：海华永泰函字（2026）第001号

### 事实部分

- 按**客观时间顺序**陈述
- **不加主观评论**，只陈述事实
- 关键时间节点要明确

### 要求部分

- 要求必须**具体、明确**
- 金额要写明具体数字
- 期限要写明具体日期
- 多项要求逐条列出

### 后果告知

- 应当**具体且合理**
- 不得威胁恐吓
- 常见表述："将依法采取诉讼/仲裁等措施"

### 注意事项

- 律师函需加盖律所公章
- 发送方式建议用EMS或公证送达
- 保留发送凭证
