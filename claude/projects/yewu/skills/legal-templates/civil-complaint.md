# 民事起诉状模板

## 文书结构

```
民事起诉状

原告：[姓名/名称]，[性别]，[出生日期]，[民族]，
住所地：[地址]
统一社会信用代码/身份证号：[代码/号码]
法定代表人：[姓名]，[职务]
委托诉讼代理人：[姓名]，[律所名称]律师

被告：[姓名/名称]，[性别]，[出生日期]，[民族]，
住所地：[地址]
统一社会信用代码/身份证号：[代码/号码]

（如有第三人）
第三人：[姓名/名称]...

诉讼请求：

一、判令被告[具体请求]；
二、判令被告承担本案全部诉讼费用。

（暂计标的额：人民币[金额]元）

事实与理由：

[当事人关系/合同签订情况]

[合同核心条款]

[合同履行情况]

[违约事实/争议事实]

[催告/协商情况]

综上所述，原告依据《中华人民共和国民法典》第XXX条等相关规定，诉至贵院，望判如所请！

此致
[法院名称]

具状人：[原告名称]
[日期：年 月 日]
```

## 关键规范

### 诉讼请求

- **暂计标的额**：在请求后以括号标注
- **违约金四要素**：按"基数→利率→起算→截止→暂计"顺序表述
  - 例：以XX元为基数，自XXXX年XX月XX日起按全国银行间同业拆借中心公布的一年期贷款市场报价利率（LPR）的X倍计算至实际清偿之日止，暂计至XXXX年XX月XX日为XX元
- **LPR 统一用小写形式**
- **诉讼费**作为独立一项列为最后请求
- **优先排列**：给付之诉 > 确认之诉 > 形成之诉

### 事实与理由（三段式）

1. **签约**：时间 + 合同名称 + 核心条款摘要
2. **履行**："上述合同签订后，……" + 具体履约细节
3. **违约+催告**：违约事实 → 催告经过 → "故诉至贵院，望判如所请"

### 注意事项

- 起诉状中法条引用宜少不宜多
- 案由应在提交前确认（参考案由速查）
- 管辖法院应确认级别管辖和地域管辖
- 保留证据原件以备核查
