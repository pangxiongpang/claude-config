# 法条核实

## 功能

核实法条是否现行有效，数据来源：国家法律法规数据库（全国人大官网）

## 与「法律检索」的分工

| 技能 | 职责 | 数据源 |
|------|------|--------|
| **法律检索** | 查找法条原文、搜索类似案例 | 威科先行 |
| **法条核实**（本 skill） | 核实在用法条是否现行有效 | 国家法律法规数据库 |

> 如需查找法条原文或搜索类似案例，调用 **法律检索** skill。

## 使用场景

- 分析案子时，自动核实引用的法条是否现行有效
- 用户直接要求核实某个法条

## 核实流程

1. **获取法条信息**：从对话中提取法条名称和条号
2. **访问数据库**：Playwright 自动化进入 flk.npc.gov.cn → 输入搜索词 → 按Enter打开新标签页 → 读取搜索结果
3. **返回结果**：现行有效 / 已修改 / 已废止

## 执行命令

```bash
# 核实法条是否现行有效
# 名称和条号分开传，脚本会自动拼成"名称第X条"（不加空格）
# ✅ 全称和简称都可以：
python D:/AI/1.业务/.claude/skills/law-validation/law_validate.py "工伤保险条例" "17"
python D:/AI/1.业务/.claude/skills/law-validation/law_validate.py "中华人民共和国劳动合同法" "7"
python D:/AI/1.业务/.claude/skills/law-validation/law_validate.py "民法典" "577"
```

## 输出格式

```
核实: 工伤保险条例 第17条
状态: ✅ 现行有效
匹配: 工伤保险条例有效 公布日期：2010-12-20 行政法规国务院

条文: 请用北大法宝 MCP 获取第17条具体内容
```

## 硬性约束

- **必须核实**：分析案子提到法条时，必须自动核实
- **权威来源**：只使用国家法律法规数据库（全国人大官网）
- **简洁输出**：只返回核实结果，不返回完整目录
- **省 token**：避免返回大量无关内容

## 注意事项

- 搜索词由脚本自动拼接：名称 + 第 + 条号 + 条（中间无空格）
- 脚本优先匹配国家级法规（不含省/市/自治区等地域名称）
- 条文具体内容用 **北大法宝 MCP** 获取（本脚本只核实时效性）
- 部分较旧的法规（找不到时）可尝试去掉"中华人民共和国"前缀
- 需要 Playwright Chromium 浏览器（已通过 MCP 安装）
