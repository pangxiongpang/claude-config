# 判决后

## 目标

判决分析 → 上诉评估 → 执行预备。

## 工作流

1. **判决分析**：
   - 提炼三段：认定事实、裁判理由、法律适用
   - 预期 vs 实际结果对比表
2. **上诉必要性评估**：
   - 改判可能性分析
   - 上诉成本估算（时间 + 费用）
   - 客户意愿确认
3. **如上诉**：
   - 委派 `legal-templates` 起草上诉状
   - 委派 `law-validation` 核实上诉引用的法条
4. **如不上诉**：
   - 执行阶段预备（财产线索排查清单）

## 产出

- [ ] 判决分析报告
- [ ] 上诉/不上诉建议
- [ ] （如上诉）上诉状
- [ ] （如不上诉）财产线索排查清单
