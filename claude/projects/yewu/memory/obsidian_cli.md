2026-07-10 经全面测试，官方 `obsidian` CLI（kepano/obsidian-skills）对中文支持良好，包括：
- 创建含中文内容的笔记
- 读取中文内容（无乱码）
- 中文关键词搜索
- 中文文件名重命名
- 中文属性名/值读写
- Prepend/Append 中文内容

已确认可用后从 `cli-anything-obsidian` 切换过来。

命令格式：`obsidian vault="胖胖熊" <command> <args>`
